# Challenge Me - Personal Challenge Manager

A Nuxt 3 application for creating and managing personal challenges.

## Features

- Create personal challenges with title, description, category, and difficulty
- Track progress with percentage completion
- Update challenge status (not-started, in-progress, completed)
- Delete challenges
- Persistent storage using localStorage

## Setup

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

## Usage

1. **View Challenges**: Main page displays all your challenges with progress tracking
2. **Create Challenge**: Click "New Challenge" to add a new personal challenge
3. **Update Progress**: Click "Update" on any challenge to modify progress (0-100%)
4. **Delete Challenge**: Remove challenges you no longer need

## Tech Stack

- Nuxt 3
- TypeScript
- Tailwind CSS
- Vue 3 Composition API
